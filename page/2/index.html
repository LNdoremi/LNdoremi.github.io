自动标注和错误类型评估——语法错误纠正

论文地址：https://www.aclweb.org/anthology/P17-1074
github地址：https://github.com/chrisjbryant/errant

better to know
edits → 编辑的内容
linguistically-enhanced alignment algorithm → 语言学增强对齐算法
Gold Reference → 专家批注
Auto Reference → 自动批注

摘要

Errant是一个语法错误标注工具，可以从平行语料（原句，纠正过的句子）中自动提取edits，并进行分类。方法包含两个主要步骤：

  1. 通过语言学增强对齐算法找出平行语料间的edits；
  2. 通过新的，基于规则的框架，该框架仅仅依赖数据集不可知的信息（词根、词性标注），将edits进行分类。

编辑提取

自动标注的第一步是编辑提取（edits extraction）。

Felice提出了一种新方法来提取edits，融合了一套合并规则的语言学增强对齐算法。更特别的是，融入了不同的语言学信息（比如词性标注和词根）到编辑距离的代价函数中，使得具有相似语言学属性的token能对齐。相较于传统的编辑距离，对齐效果更好。

自动错误分类

  1. 基于规则的错误分类框架
    根据词性标注，编写了50条规则来进行分类。错误类型主要分为25种。

  2. 数据集未知的分类器
    基于规则的方法一个关键优势是：独立于数据集，完全不需要标注的训练数据。第二个显著的优势是：这个方法能准确地将edits分到特定的类别。相反，人类和机器学习的分类决策没有这么透明。

  3. 自动标注
    使用到以下工具：
    spaCy: 词性标注、词性还原、句法依存解析
    NLTK: 词干提取
    Hunspell GB-large word list: non-word errors

  4. 错误类别
    错误类别有25种，前缀有：’M’, ‘R’, ‘U’。通过前缀和后缀的组合，能从不同的层面考察系统的表现。
    ‘M’: Missing
    ‘R’: Replacement
    ‘U’: Unnecessary

错误类型得分

  1. Gold Reference VS Auto Reference
    在CoNLL-2014 共享任务中进行测评，结果显示，自动批注质量和专家批注效果一样好。

  2. 与M2评分对比
    M2评分偏高

CoNLL-2014 共享任务分析

选择CoNLL-2014 共享任务的数据来测试ERRANT工具，因为该任务收集了最多的GEC系统生成的公开数据。

结论

ERRANT为GEC错误类型评估提供便利，并且将现存的错误纠正语料标准化，减少标注的工作量。

What is entropy?

Information entropy is the average rate at which information is produced by a stochastic source of data.
信息熵是衡量一组随机数据产生的信息的度量值。
The basic idea of information theory is that the “news value” of a communicated message depends on the degree to which the content of the message is surprising.
信息熵的基本思想是：一则消息的“新闻价值”取决于消息内容让人吃惊的程度。
If an event is very probable, it is no surprise (and generally uninteresting) when that event happens as expected.
如果一个事件很容易发生，当它按照预期发生时，一点也不惊奇（也不有趣）。
However, if an event is unlikely to occur, it is much more informative to learn that event happened or will happen.
然而，如果一个事件不太容易发生，当知道它要发生时，包含的信息量更大。
For instance, the knowledge that some particular number will not be the winning number of a lottery provides very little information, because any particular chosen number will almost certainly not win.
举个例子，一些特定的数字不会成为彩票的获奖号码，这个消息提供了很少的信息，因为任何一个特定的号码都不一定会赢。
However, knowledge that a particular number will win a lottery has high value because it communicates the outcome of a very low probability event.
然而，一个特定的号码将赢得彩票，这个消息有很高的价值，因为它传达出一个概率非常低的结果。

What is information gain?

In information theory and machine learning, information gain is the amount of information gained about a random variable or signal from observing another random variable.
在信息论和机器学习中，信息增益是通过观察其它的随机变量来获得对于某一随机变量（信号）的信息。
